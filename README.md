# 📱 Smart Irondo System

**Smart Irondo System** is a modern, digital platform designed to **empower Abanyerondo** by digitizing their daily work processes. The system enables users to submit daily reports, upload evidence, request support, and perform their tasks efficiently — all while being fully managed by an admin.  

The project includes:  
- **Backend API** with Node.js, Express, Drizzle ORM, and PostgreSQL  
- **Admin Panel** built with React.js and Tailwind CSS  
- **Mobile App** for users built with Expo (React Native), supporting push notifications  

---

## 🌟 Key Features

### **User Workflow (Mobile App)**
- **Daily Reports**: Users can submit daily activities.  
- **Evidence Upload**: Attach images or files as proof of work.  
- **Support Requests**: Ask for help from admin to perform tasks effectively.  
- **Activation via SMS**: Admin registers the user → SMS activation code is sent → user verifies code → sets password.  
- **Push Notifications**: Users receive alerts for approvals, support responses, and reminders.  

### **Admin Panel**
- **User Management**: Register new users, manage activations, track user activity.  
- **Support Management**: Respond to user requests and monitor support tickets.  
- **Report Tracking**: View all submitted reports and evidence.  
- **Activity Overview**: Analytics and monitoring of daily work performance.  

---

## 🛠 Technologies Used

- **Backend**: Node.js, Express.js, Drizzle ORM, PostgreSQL  
- **Admin Panel**: React.js, Tailwind CSS  
- **Mobile App**: React Native with Expo Framework  
- **Notifications**: Push notifications for mobile users  
- **Authentication**: Admin-managed registration and SMS-based activation
